# usage guides: 
# First, the user sees the main menu. You can first add clients( by going to manage clients first), and add projects( by going to manage projects first). You can only create an invoice after adding at least one client and one project.
# After creating projects, you can go to manage projects and view the status of the projects by selecting project financial breakdown.
# For projects, you can log expenses or hours worked
# in the main menu, you can also view financial reports and project status chart.
# in invoices, there is an option to view invoice in client currency. This gives the user the ability to view the amount in the currency they will be paid in.
